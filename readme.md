## Dataset

https://www.kaggle.com/competitions/state-farm-distracted-driver-detection

The dataset consists of nine different distraction classes, including activities such as texting-right, talking on the phone-right, texting-left, talking on the phone-left, operating the radio, drinking, reaching behind", hair and makeup", and talking to the passenger. The dataset contains 15.200 labeled train images and 3.040 test images.
